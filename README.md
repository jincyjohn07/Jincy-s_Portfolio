## üçï Pizza Sales Data Analytics Project

This project is an **end-to-end data analytics project** designed to analyze pizza sales data and evaluate overall business performance. The main goal of the project is to convert raw sales data into **clear, meaningful insights** that support better business decision-making.

The analysis focuses on understanding revenue, customer ordering behavior, and product performance through key performance indicators (KPIs) and visual reports.

---

### Project Objective
The objective of this project is to:
- Monitor key sales performance metrics  
- Understand customer preferences and ordering patterns  
- Identify best-selling and underperforming pizza products  
- Support data-driven business decisions using simple and clear insights  

---

### Tech Stack
The following tools and technologies were used to build this project:

- **Power BI** ‚Äì Interactive dashboards and KPI visualization  
- **Tableau** ‚Äì Advanced data visualization and storytelling  
- **SQL** ‚Äì Data extraction, joins, filtering, and KPI calculations  
- **Python** ‚Äì Data cleaning, transformation, and analysis  
- **Excel** ‚Äì Initial data exploration and validation  

---

### Key Performance Indicators (KPIs)
The following KPIs are calculated to measure business performance:

- **Total Revenue** ‚Äì The total income generated from all pizza sales  
- **Average Order Value** ‚Äì The average amount spent per order  
- **Total Orders** ‚Äì The total number of customer orders  
- **Total Pizzas Sold** ‚Äì The total quantity of pizzas sold  
- **Average Pizzas per Order** ‚Äì The average number of pizzas per order  

---

### Data Analysis & Visualizations
To make insights easy to understand, the project includes multiple visualizations:

- **Sales Percentage by Pizza Size**  
- **Total Pizzas Sold by Category**  
- **Top 5 Best-Selling Pizzas** (by Revenue, Quantity, Orders)  
- **Bottom 5 Worst-Selling Pizzas** (by Revenue, Quantity, Orders)
